## Pricing Breakdown

**All prices are per 1 million tokens** (≈750 pages of text)

| Token type | Price | When does this happen? |
|------------|-------|------------------------|
| **Input** (reading) | $3.00 | When AI reads your prompt or code |
| **Cache Write** (saving) | $3.75 | When AI stores info for later reuse |
| **Cache Read** (retrieving) | $0.30 | When AI retrieves saved info (cheaper!) |
| **Output** (writing) | $15.00 | When AI generates code/response |

**Important note:** Output (writing) is **5x more expensive** than input (reading). That's because generating new content is harder than reading existing content.
