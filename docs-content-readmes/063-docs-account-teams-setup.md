This is the **Teams & Enterprise Setup** documentation – it explains how to create and set up a **Cursor Teams plan** for organizations.

Think of this as the **admin guide** for getting your team started with Cursor – creating the team, inviting members, setting up SSO, and managing billing.

Let me break this down.

---

## Overview

**Cursor works for individuals and teams. The Teams plan provides tools for organizations: SSO, team management, access controls, and usage analytics.**

| Individual Plan | Teams Plan |
|-----------------|------------|
| Single user | Multiple users |
| No team management | Team management dashboard |
| No SSO | SSO support |
| No usage analytics | Usage analytics |
| No access controls | Access controls |

---

## Creating a Team

### Step 1: Set Up Teams Plan

| User Type | How to start |
|-----------|--------------|
| **New users** | Visit `cursor.com/team/new-team` to create a new account and team |
| **Existing users** | Go to your dashboard and click **"Upgrade to Teams"** |

### Step 2: Enter Team Details

- Select a **Team name**
- Select a **billing cycle** (monthly or annual)

### Step 3: Invite Members

| Feature | Description |
|---------|-------------|
| **Invite members** | Send invites to team members |
| **Prorated billing** | User counts are prorated – you only pay for the time users are members |
| **Domain matching** | Opt in so teammates with verified, matching email domains can join your team without an invite. Configure in team settings. |

### Step 4: Enable SSO (Optional)

> *"Enable SSO for security and automated onboarding."*

SSO (Single Sign-On) allows team members to log in using your organization's identity provider.

## Common Beginner Questions

### Q: How do I create a team if I already have a Cursor account?
**A:** Go to your dashboard and click **"Upgrade to Teams"**.

### Q: Do I need to pay for a full year upfront?
**A:** No – you can choose monthly or annual billing cycles. User counts are prorated.

### Q: Can team members join without invites?
**A:** Yes – enable **domain matching** in team settings. Teammates with verified email addresses from your domain can join automatically.

### Q: Does the Teams plan include SSO?
**A:** Yes – SSO is available as an optional feature for security and automated onboarding.

### Q: Can I be on multiple teams?
**A:** Yes – you can be a member of more than one team.

### Q: What if my company uses a proxy or VPN?
**A:** Cursor should work. You may need to configure network settings (see Network Configuration documentation).

## The Bottom Line

**Setting up a Cursor Teams plan is straightforward:**

1. **New users** → `cursor.com/team/new-team`
2. **Existing users** → Dashboard → "Upgrade to Teams"
3. Choose team name and billing cycle
4. Invite members (or use domain matching for auto-join)
5. Optionally enable SSO

**Key benefits of Teams plan:**
- Centralized team management
- SSO for security
- Usage analytics
- Access controls
- Prorated billing (pay only for time users are members)

**For admin setup:** Start with step 1 above, then configure additional settings in your team dashboard.
